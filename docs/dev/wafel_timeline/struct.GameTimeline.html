pub struct GameTimeline<M: GameMemory, C: GameController<M>> { /* fields omitted */ }
Expand description

An implementation of the Wafel timeline algorithm.

This struct wraps a GameMemory and a GameController and provides the ability to request the game state on arbitrary frames using the frame method.
